## Brand Style Guide (Baked into all prompts)

### Lumicello Visual Identity
- **Primary Colors**: Sage green (#6B7B5F), Warm gold (#C4A962), Soft cream (#F5F1E8)
- **Secondary Colors**: Coral (#E8A87C), Teal (#7BA5A5), Soft pink (#E8C4C4), Warm yellow (#E8D4A8)
- **Style**: Soft, organic, nurturing, Scandinavian-inspired minimalism
- **Mood**: Warm, gentle, developmental, nature-connected
- **Typography feel**: Rounded, friendly, approachable

### Illustration Style Keywords
```
soft watercolor texture, organic rounded shapes, gentle gradients,
warm muted palette, Scandinavian children's book illustration style,
hand-drawn feel, nurturing aesthetic, nature-inspired,
minimal clean composition, educational toy aesthetic
```

## Output Specifications

| Asset Type | Format | Dimensions | Notes |
|------------|--------|------------|-------|
| Illustrations | SVG preferred, PNG fallback | Variable | Export at 2x for retina |
| Box photos | WebP | 800x800 | Square crop, white background |
| Flat-lay photos | WebP | 1200x800 | 3:2 ratio, styled arrangement |
| Icons | SVG | 64x64 base | Scalable vector |
| Patterns | SVG | Tileable | Seamless repeat |
