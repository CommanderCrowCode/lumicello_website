### How to Get Kit.com Form Code
1. Log in to [Kit.com](https://kit.com)
2. Go to **Grow** → **Landing Pages & Forms**
3. Create or select your newsletter form
4. Click **Publish** → **HTML**
5. Copy the form action URL or full embed code
