## Visual Language

### Typography

| Element | Font | Weight | Size |
|---------|------|--------|------|
| Hero Title | Fraunces | 700 | 4-6rem |
| Section Titles | Fraunces | 600 | 2.5-3rem |
| Phase Titles | Fraunces | 500 | 2rem |

| Captions | Nunito | 300 | 0.875rem |
| Thai Text | Noto Sans Thai | 400 | 1rem |

### Animation Principles

1. **Smooth & Natural** - Ease-out curves, natural motion
2. **Purposeful** - Every animation communicates something
3. **Subtle** - Support the content, don't distract
4. **Performance** - Use transforms and opacity only
5. **Reduced Motion** - Respect `prefers-reduced-motion`

### Key Animations

| Animation | Trigger | Duration | Easing |
|-----------|---------|----------|--------|
| Tree growth | Scroll position | Continuous | Linear |
| Content reveal | Intersection Observer | 0.8s | Cubic-bezier(0.16, 1, 0.3, 1) |
| Box opening | Scroll + threshold | 0.6s | Ease-out |
| Items float out | Box open complete | 0.4s stagger | Ease-out |
| Background shift | Section change | 1.2s | Ease-in-out |
| Confetti | Birthday section | 3s | Physics-based |

### Scroll Behavior

| Technique | Use Case |
|-----------|----------|
| `position: sticky` | Pinned visuals during content scroll |
| Intersection Observer | Trigger reveal animations |
| Scroll progress | Tree growth, timeline fill |
| Scroll snap (optional) | Section alignment on mobile |

### Performance Considerations

1. **Use CSS transforms only** - Avoid layout-triggering properties
2. **Debounce scroll handlers** - Limit calculations per frame
3. **Lazy load images** - Load as sections approach viewport
4. **Preload critical assets** - Hero images, tree SVG
5. **Use `will-change` sparingly** - Only on animated elements
6. **Test on low-end devices** - Ensure smooth 60fps

| Canvas (confetti) | CSS-only celebration |
